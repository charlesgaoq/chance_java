数据导入接口


导入前要首先将文件上传到服务器指定的目录（暂时/sdbimprt）下，然后在发送请求信息！
请求成功后会返回任务记录的_id及存储的位置信息！可以通过查询接口查看导入信息！
数据导入成功后对应的文件会更名为.ok结尾的文件，失败后会更名为.error结尾的文件！


请求信息：http://localhost:8080/database-web/_eversec/_sdbimprt?cs=ever&cl=ti&pfile=a.txt&cmd=json
返回信息： 

{
 "cmd":"json",
 "code":0,
 "total":1,
 "datas":[
           {
	         "taskCS":"evertask",
	         "_id":"56309f31a4f6173b98a0bda3",
	         "taskCL":"imprttask"
           }
         ],
 "exceptions":""
}

cmd:数据格式
code：0正常，其它错误
total：1（无意义）
taskCS:导入记录信息集合空间
taskCL:导入记录信息集合
_id：导入记录信息id
exceptions:异常错误信息，成功为空串

导入信息：查询taskCS，taskCL对应_id的内容 示例： 
	{ 
		"_id": { "$oid": "563206b9e4b087406d2618ab" },
		"cl": "ti",		目标集合
		"cs": "ever",	目标集合空间
		"dir": "/sdbimprt/",	数据文件地址
		"message": "",			异常错误信息
		"pfile": "a.txt",		文件名
		"state": "2",			0：等待执行；1：执行中；2执行完毕
		"tasktime": "18" 		耗时(ms)
	}
