impala 数据接口


请求信息：http://localhost:8080/database-web/_eversec/_idbsql?cmd=query&command=select source from cncert_botortrojan_event
cmd:请求类别，[query]，目前支持query操作
command:请求查询语句，程序调用时最好将参数值encode编码转义

注意以上两个参数为必须参数
返回信息： 

	{
	 "cmd":"query",
	 "code":0,
	 "total":1,
	 "datas":[{},{},... ],
	 "exceptions":""
	}
	
	cmd:数据格式
	code：0正常，其它错误
	total：1 数据记录数
	datas：返回的结果集
	exceptions:异常错误信息，成功为空串
